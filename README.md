## Cloning
```sh
git clone https://github.com/progsysdriver/botol-bot
cd botol-bot
```

## Installation Termux or Ubuntu/Debian

```sh
pkg install -y ffmpeg imagemagick libwebp
wget https://git.io/webp2gif && chmod +x webp2gif
npm install
```

## Run Bot
```sh
node index.js
```
